# EatDaBurger

A burger logger with MySQL, Node, Express, Handlebars and a homemade ORM. MVC design pattern if followed; used Node and MySQL to query and route data in the app, and Handlebars to generate the HTML.

## Technologies Used:
  * MySQL
  * Node.js
  * Express
  * Handlebars
  * ORM
